# Remotion 教学短片

## 定位
- 用途：自媒体口播视频上方的引导特效叠加
- 风格：程序员教学风 + 赛博朋克美学
- 目标：10 秒内教会观众一个知识点，建立"AI 知识博主"人设

## 基础参数
- 时长：9秒（270帧）
- fps：30
- 分辨率：1920×1080
- 输出：ProRes 4444 .mov（透明背景 + Alpha）

## 分镜脚本

### Hook（0–1.5s / 0–45帧）
"React 只能做网页？"
- 巨大问号从中心缩放弹出
- 文字带故障 RGB 分离效果
- 目的：制造认知冲突，吸引停留

### 教学演示（1.5–6s / 45–180帧）
三段式代码教学，每段代码 → 对应视觉效果：

**第1段：`<Composition />` 定义视频（45–90帧）**
- 左侧出现代码行，右侧出现视频画幅框
- 展示 Composition = 定义分辨率、帧率、时长

**第2段：`useCurrentFrame()` 驱动动画（90–135帧）**
- 代码行切换到 useCurrentFrame
- 中央几何图形开始旋转、脉冲
- 展示"当前帧号 → 画面状态"的映射关系

**第3段：`interpolate()` 平滑过渡（135–180帧）**
- 代码行切换到 interpolate
- 粒子从四周汇聚到中心
- 展示"数值映射实现丝滑动画"

### 结论（6–7.5s / 180–225帧）
"Remotion：程序员的视频引擎"
- 主标题出现，全息渐变
- 所有视觉元素达到峰值

### 收尾（7.5–9s / 225–270帧）
- 画面元素依次淡出
- 粒子最后消散
