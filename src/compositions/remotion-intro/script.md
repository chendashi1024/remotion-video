# Remotion 入门介绍

## 基础参数
- 时长：9秒（270帧）
- fps：30
- 分辨率：1920×1080
- 风格：暗黑 + 霓虹发光 + 粒子

## 分镜

### 镜头1：暗场开启（0–0.5s）
- 纯黑背景渐亮，网格浮现

### 镜头2：关键词飞入（0.5–2s）
- "React" 从左侧飞入 → 左上区域
- "组件" 从右侧飞入 → 中央偏下
- "渲染" 从右上方飞入 → 右上区域
- 每个词带霓虹发光效果， stagger 15帧

### 镜头3：几何图形构建（2–4.5s）
- 中央出现双环旋转（正反向）
- 六边形从无到有
- 中心光点脉冲

### 镜头4：主标题（4.5–6.5s）
- "Remotion" 大字，蓝紫青渐变色
- text-shadow 发光

### 镜头5：副标题 + 代码（6.5–7.5s）
- "用 React 创作视频" 副标题
- 底部代码行：`import { useCurrentFrame } from "remotion";`
- 闪烁光标

### 镜头6：收尾（7.5–9s）
- 粒子向外扩散
- 蓝光闪烁
- 全屏淡出到黑
